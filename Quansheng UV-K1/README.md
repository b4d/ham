# Quansheng UV-K1 (Mini-Kong, UV-K1(8))

## Cheatsheet

- Button 1 short press - VFO A/B
- Button 1 long press - Scan
- Button 2 short press - Power select
- Button 2 long press - Monitor
- Button MENU long press - RX mode

## Programming

Use CHIRP with installed module from https://github.com/armel/uv-k1-k5v3-firmware-custom/releases.

> USB-C cable


## Flashing firmware

https://github.com/armel/uv-k1-k5v3-firmware-custom/releases

Flash via web browser.

Hold PTT and turn on the radio to enter DFU mode, flashlight will go ON.

## Settings
Open hidden menu:
Hold the PTT + F1 (button below ptt) together and turn the radio on.

### 75 BatTyp
Set to `1400mAh K1` or `2500mAh K1`, affects only % not voltage.

### 76 SetNav
Set to `Left Right UV-K1`.
